# @saber2pr/gen-less

> less-bundler.

```bash
# from npm
npm install @saber2pr/gen-less -g

git clone https://github.com/Saber2pr/gen-less.git
```

# Usage

```bash
genLess ./src

genLess ./src ./output.css

npx @saber2pr/gen-less ./src
```

---

## start

```bash
npm install
```

```bash
npm start

npm test

```
